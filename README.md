## Getting Started

### Installation

- with NPM
  
  ```$ npm install -g open-source-npm-package-template``` 
- with Yarn 
  
  ```$ yarn global add open-source-npm-package-template```

### Usage

**Run an app with**
```
$ open-source-npm-package-template
```
